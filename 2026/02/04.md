-「特定の処理が実行された後」ではなく、**「画面（見た目）が書き換わった（レンダリングされた）後」**に動く処理
もっと言うと、**「今の状態（State）に合わせて、画面の外の世界（APIやDOMなど）を同期させる機能」**と考える

```js
export const getAllPokemon = (url) => {
  return new Promise((resolve, reject) => {
    fetch(url)
      .then((res) => res.json())
      .then((data) => resolve(data));
  });
};

resolve : 成功した時
reject：失敗した時
```

- Promiseは、fetchの処理が全て成功するまで待ってくださいね〜という意味
