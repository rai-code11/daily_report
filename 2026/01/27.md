## わかったこと
async「この中では時間がかかる処理（=通信など）を待てますよ」という合図。
- ←fetch はネット通信で時間がかかるから async が必要になる

```js
  const allChar = await fetch(
    "https://ihatov08.github.io/kimetsu_api/api/all.json",
  );
```

- fetchはそのURLにアクセスしてデータを取りに行く命令
- fetchはそのURLにアクセスしてデータを取りに行く命令
- awaitがないとまだデータが返ってきていないのに、次の行に行ってしまいバグる

```js
const data = await allChar.json();
```

- allChar.json() は「返ってきた内容をJSONとして読み取って、JavaScriptで使える形（配列やオブジェクト）に変換する命令
- ←これも時間がかかることがあるためawaitで待っている


- fetch：お店（URL）に注文しに行く
- await：届くまで待つ
- .json()：届いた荷物を開けて中身を使える形にする
