## わかったこと

- argumentsには実引数が渡ってくる
- arguments[0]=引数の1番目
- arguments[1]=引数の2番目

fn(…args):配列になった引数を入れることができる

```js
const formattedWeekdays = weekdays.map((d) => "  " + d);
```
配列から新しい配列を作成するのに便利
- weekdays[array]に対してmapを使用し変数dにはarray.valueが格納され、”  ” + dに変換されて新しい配列が作られる

【for文の場合】
* let arr = [] と、まず空の箱を用意しなければならない。
* i < weekdays.length など、終わりの条件を自分で書く必要がある,書き間違えるとバグる。
* arr.push() で自分で箱に入れなければならない。
【.map() の場合】
* 「何をするか（空白を足す）」だけに集中して書ける。
* 空の箱の用意や、入れ直す作業はJavaScriptが裏側でやってくれる。
