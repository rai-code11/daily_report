## わかったこと

同期処理
- １つの処理が完了するまで次の処理には進まない
- メインスレッドでコードが順番に実行される

非同期処理
- 一時的にメインスレッドから処理が切り離される
非同期処理機能
- 非同期API（setTimeout,Promise,queueMicrotask）
- UIイベント
- クリック
- NWイベント
- I/Oイベント

タスクキュー
- 実行待ちの非同期処理の行列
- キューはところてん
キューの仕組みを「先出し先入れ」という

コールスタック関数
```js
function a() {
  b();
}

function b() {
  c();
}

function c() {
  console.log("Hello");
}

a();
```
関数の呼び出しチェーン（入れ子） そのとき裏で使われている箱（仕組み）がコールスタック
