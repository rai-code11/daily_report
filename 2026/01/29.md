A && B は、
* A が falsy なら A を返す（そこで止まる）
* A が truthy なら B を返す

```js
0 && "B"      // 0（最初がfalsyなので0を返して終わり）
"" && "B"     // ""（空文字もfalsy）
12 && "B"     // "B"（12はtruthyなので右を返す）
"ok" && 5     // 5（左truthyなので右を返す）
```
