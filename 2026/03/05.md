## わかったこと
```js
 {role === 'all' ? '全員' : role === 'student' ? '生徒のみ' : 'メンターのみ'}
```
「三項演算子」を入れ子（ネスト）にして、「もし〇〇ならA、そうでなくて△△ならB、それ以外ならC」**という条件分岐を1行で書いたもの

カスタムフックはただの関数
1. 投げる（App側）
フックという「関数」を呼び出し、必要な材料を引数として投げ込む。
```JavaScript

const { handleRegister } = useUserRegistrationForm(allUsers, setAllUsers);

```
2. 料理する（Hooks側）
渡された材料（allUsersなど）を使って、フィルタリングしたり、バリデーションしたり、useStateを動かしたりする。
3. 返す（Hooks側）
App側で「ボタンのクリック」や「テーブルの表示」に使いたいものを、一つのオブジェクトにまとめてreturnする。
```JavaScript

return { newUser, onChangeNewUser, handleRegister };
```
4. 受け取る（App側）
返ってきたオブジェクトを分割代入でパッと展開して、変数として使えるようにする。
