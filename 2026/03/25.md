## わかったこと
**not-found.tsx**
- 存在しないページにアクセスされた際に表示される404エラーページを定義するファイル

**route.ts**
- APIエンドポイントを定義するファイル

**middleware.ts**
- リクエストが処理される前に実行される中間処理を定義するファイル

**public**
- 静的ファイルを設置するためのディレクトリ

```
public/logo.png**
→ 
<image src="/logo.png" />
```

**Dynamic Route Segments**
- 動的なルートパラメータを持つページを作成するための機能
```
app/users/[id]/page.tsx
→
/users/1
→
/users/[id]/page.tsxの内容

idとして1を取得することができる
```
**Route Groups**
- URLのパスに影響を与えることなく、ルートをグループ化するための機能
```
app/(auth)/users/page.tsx
→
/usersで下記のページが表示される
/(auth)/users/page.tsx
```
