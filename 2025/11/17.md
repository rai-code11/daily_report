###  1. 基本の使い方

#### ◆ padStart

```js
文字列.padStart(最終的な長さ, 追加する文字);
```

- 左側（先頭）に最終的な長さになるまで文字を足す  
- 第2引数は省略可（省略すると `" "`（半角スペース）が使われる）

**例：ゼロ埋め（よくやるやつ）**

```js
const num = "7";

console.log(num.padStart(3, "0")); // "007"
console.log(num.padStart(5, "0")); // "00007"
```

---

#### ◆ padEnd

```js
文字列.padEnd(最終的な長さ, 追加する文字);
```

- 右側（末尾）に最終的な長さになるまで文字を足す

**例：表示をそろえるときに使う**

```js
const name = "Apple";

console.log(name.padEnd(10, ".")); // "Apple....."
console.log(name.padEnd(10));      // "Apple     "（スペースで埋まる）
```

#### 共通設定
追加したい文字がない場合、第２引数のみ省略できる
全部スペースになる
