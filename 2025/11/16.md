### models の `blank=True` 

#### 【概要】
`blank=True` は **「フォームで空欄を許可する」** ための設定。  
Django の **バリデーション（入力チェック）** に影響する。

---

#### 【作用する場所】
- Django 管理画面（Admin）
- ModelForm / Form
- ユーザー入力フォーム

→ **空欄のまま送信してもエラーにならない**

---

### 【blank=True と null=True の違い】
```
| 役割 | blank=True | null=True |
|------|------------|-----------|
| フォーム入力 | 空欄を許可する | 影響なし |
| DBでの扱い | 空欄は `""`（空文字）として保存 | DBの値として `NULL` を許可する |
```
● 文字系フィールド（CharField / TextField）  
　→ 普通は **blank=True のみ**

● 数値 / 外部キーなど  
　→ **blank=True + null=True をセットで使う**

---

### 【コード例】

```python
class Product(models.Model):
    title = models.CharField(max_length=100, blank=True)

```

フォーム：空欄OK
DB：空文字 "" が入る
