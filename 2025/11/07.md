### JS `Array.prototype.map` 

#### 概要
- **配列の各要素を“変換”して新しい配列を返す！**
- 元の配列は**変更しない（非破壊）**。

#### 構文
```js
const newArray = array.map((value, index, array) => {
  // 返した値が newArray の要素になる
}, thisArg);
```

#### 引数
- **value**: 各要素  
- **index**: 要素のインデックス  
- **array**: 元の配列  
- **thisArg**: コールバック内の `this`（省略可）

#### 基本例
```js
[1,2,3].map(n => n * 2);                 // [2,4,6]
["1","2","3"].map(Number);               // [1,2,3]
["a","b"].map((v,i) => `${i}:${v}`);     // ["0:a","1:b"]
```
