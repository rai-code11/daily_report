### Django `post()` メソッドの基本メモ

### 目的
HTMLフォームなどから送られた **POSTリクエストのデータを受け取る**。

### 基本構文（CBV）
```python
    def post(self, request, *args, **kwargs):
        # HTMLフォームの name 属性と一致するキーで取得
        title = request.POST.get("title")
        file  = request.FILES.get("image")
        # ここでDB更新やバリデーションなど
        # return HttpResponse(...)
```
### 処理の流れ
1) `<form method="POST">` で送信  
2) Djangoが受信 → `post()` が呼ばれる  
3) 値を取得：`request.POST`（文字データ）、`request.FILES`（ファイル）  
4) 必要ならモデルに保存 → `obj.field = ...` → `obj.save()`

### 引数の意味
- `self`：ビューインスタンス  
- `request`：HTTPリクエスト（フォーム内容・ユーザー情報などを保持）  
- `*args, **kwargs`：URLで渡された値（例：`<int:pk>` は `kwargs["pk"]` で取得）

### 注意点
- CSRF対策：フォーム内に `{% csrf_token %}`  
- ファイル送信：`<form enctype="multipart/form-data">`  
- JSON送信時：`request.POST` ではなく `json.loads(request.body)` を使う

### まとめ
- `post(self, request, *args, **kwargs)` は **HTMLから送られたデータを受け取る入口**。  
- フォーム値は `request.POST`、ファイルは `request.FILES`。  
- URLの `<int:pk>` などは `kwargs` から取得し、必要に応じてDBへ保存する。
