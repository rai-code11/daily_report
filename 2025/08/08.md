### 1. 文字列（Strings）

- `" "` または `' '` で文字列を作成できる
```python
print("Hello world!!")
print('Hello world!!')
```

- `'` を含む場合は `"` または `\'` でエスケープする
```python
print("I'm fine")
print('I\'m fine')
```

- 複数行は `""" """` または `''' '''` で囲む
```python
print("""
Hello world!!
How are you doing?
""")
```

- 改行 `\n` やタブ `\t` を挿入できる
```python
print("break\nline")
print("tab\tinserted!!")
```

- `\\` でバックスラッシュ、`\'` / `\"` でクォーテーションをエスケープ
```python
print("not break\\nline")
print("not tab\\tinserted!!")
```

- `+` で文字列を連結できる
```python
print("hello" + " world" + "!!")
```

# 関数のイメージ：汁（環境変数）＋カレー粉（引数）でカレーを完成させる

### 3. 変数・型

- 変数に値を代入できる
```python
hello = 'hello'
world = 'world'
print(hello)
print(world)
```

- 文字列埋め込みは `format()` を使える
```python
print("{} {}".format(hello, world))
```

- 文字列埋め込みは f-string も使える
```python
name = "John"
print(f"Hello, {name}")
```

- `type()` で型確認ができる
```python
hello_type = type("Hello")
print(hello_type)
```

- `id()` でオブジェクトIDを確認できる
```python
hello = "hello"
hello2 = "hello"
print(id(hello))
print(id(hello2))
print(id("hello"))
```

---

### 4. 数値（int / float）と演算

- 整数は `int` 型
```python
print(type(1))
```

- 小数は `float` 型
```python
print(type(0.1))
```

- 浮動小数点は誤差が出る場合がある
```python
print(0.1 + 0.1 + 0.1)
```

- 四則演算が可能
```python
print(1 + 0.4)
print(1 - 0.4)
print(2 * 3)
print(1 / 2)  # int同士でもfloatになる
```

- その他の演算子
```python
print(14 // 3)  # 整数部分
print(14 % 3)   # 余り
print(2 ** 3)   # べき乗
```

- 演算結果を変数に代入できる
```python
result = 1 + 5
print("1 + 5 = {}".format(result))
```

- 変数同士でも演算できる
```python
hit_point = 100
attack_point = 40.3
remain = hit_point - attack_point
print(f"remain hit point is {remain}")
```

- 複合代入演算子（+=, -= など）が使える
```python
a = 1
a += 1
print(a)
```

- f-stringで小数点の桁数や表示幅を指定できる
```python
value = 1/3
print(f"value is {value:.4f}")
print(f"value is {value:.2f}")
print(f"value is {value:10.2f}")
print(f"value is {value:6.2f}")
```

### 6. Boolean型・比較演算子・論理演算子

- 真偽値は `True` / `False`
```python
print(True)
print(False)
```

- 比較演算子で大小比較ができる
```python
print(5 > 3)
print(5 == 3)
print(1 == 1.0)
print(1 == "1")
```

- `is` / `is not` はオブジェクトの同一性を判定する
```python
a = 1
b = 1
print(a is b)
print(a is not b)
```

- 論理演算子 `and` / `or` / `not` が使える
```python
first_condition = True
second_condition = False
print(first_condition and second_condition)
print(first_condition or second_condition)
print(not first_condition)
```

- 短絡評価が行われる
```python
print(10 > 3 or undefined == 20)
```

### 7. if文

- `if` / `elif` / `else` で条件分岐できる
```python
age = 23
age_alcohol = 20
if age > age_alcohol:
    print("you can drink beer!")
else:
    print("you are too young to drink beer")
```

- `not` で条件を反転できる
```python
age = int(input("how old are you?"))
if not 0 < age < 120:
    print("the value is invalid!")
```

- `None` の判定は `if not a:` が推奨
```python
a = None
if not a:
    print("a is None")
```

### 8. リスト（lists）

- 複数の値を順序付きで保存できる
```python
fruits = ['apple', 'peach', 'melon', 'grapes']
print(fruits)
```

- インデックスでアクセスできる（0始まり、負数は後ろから）
```python
print(fruits[0])
print(fruits[-1])
```

- 二重リストにもアクセスできる
```python
hetero_list = ['string', 1, 3.4, True, fruits]
print(hetero_list[-1][0])
```

- `.append()` で末尾に追加
```python
fruits.append('banana')
```

- `.insert()` で位置を指定して追加
```python
fruits.insert(3, 'lemon')
```

- `.remove()` で一致する最初の要素を削除
```python
fruits.remove('peach')
```

- `.sort()` で昇順、`reverse=True` で降順に並び替え
```python
fruits.sort()
fruits.sort(reverse=True)
```

- `len()` で要素数を取得
```python
print(len(fruits))
```

- スライスで複数要素を取得
```python
even = [2, 4, 6, 8, 10, 12]
print(even[1:4])
print(even[:4])
print(even[3:])
print(even[::-1])
```

- `+` や `+=` でリストを結合
```python
print([1, 2, 3] + [4, 5, 6])
a = [1, 2, 3]
b = [4, 5, 6]
a += b
print(a)
```

- `" ".join(list)` で文字列に結合
```python
text = " ".join(['Hi', 'My', 'name', 'is', 'John'])
print(text)
```

- `.split()` で文字列をリストに分割
```python
splited_space = "Hi My name is John".split()
splited_slash = "Hi/My/name/is/John".split("/")
print(splited_space)
print(splited_slash)
```
