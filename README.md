# daily_report

学習・作業の進捗を日々記録するためのリポジトリです。
後から振り返って「何をやって、何が詰まり、次に何をするか」がわかるように記述をします。

## 日報の場所
- 日報は `xxxx(例：2025年→2025/` に保存しています

## ルール
- ファイル名: `DD.md`
- できるだけ毎日

## 日報テンプレ
```md
## 取り組んだ課題一覧
- 

## 学んだこと(Hungry)
- 

## 日常業務で改善できそうな所/属人化しない仕組み化の動き(Sustainable)
- 

## 改善案を具体的に(Sustainable)
- 

## チームに共有できそうなこと/締切意識(Owener)
- 

## 次やること
- 
- 

## 感じたこと
- 

## 学習時間
-
```
